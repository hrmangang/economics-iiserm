<h2 align="center">Shinobi</h2>

The tier list is [here](https://www.reddit.com/r/ninjavoltage/wiki/tierlist/). The following suffixes should be kept in mind.
 
- 💥: is a special shinobi trained for AOM
- 🏰: has duties on fortress
- 🌀: is the (solo) main in attack missions

Within each type, the shinobi are ordered from strongest to weakest in terms of damage.
